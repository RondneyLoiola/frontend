# 💰 Expense Tracker

Um aplicativo web moderno para controle de despesas pessoais, desenvolvido com React, TypeScript e Vite.

## 📋 Sobre o Projeto

O Expense Tracker é uma aplicação que permite aos usuários registrar, visualizar e gerenciar suas despesas de forma simples e intuitiva. Ideal para quem busca ter um melhor controle financeiro pessoal.

## ✨ Funcionalidades

- ✅ Adicionar novas despesas
- ✅ Visualizar lista de despesas
- ✅ Editar despesas existentes
- ✅ Excluir despesas
- ✅ Categorizar despesas
- ✅ Visualizar resumo financeiro
- ✅ Interface responsiva e moderna

## 🚀 Tecnologias Utilizadas

- **React** - Biblioteca JavaScript para construção de interfaces
- **TypeScript** - Superset do JavaScript que adiciona tipagem estática
- **Vite** - Build tool rápida e moderna
- **ESLint** - Ferramenta de linting para manter código limpo
- **Biome** - Linter e formatador de código

## 📦 Pré-requisitos

Antes de começar, você precisará ter instalado em sua máquina:

- [Node.js](https://nodejs.org/) (versão 18 ou superior)
- [npm](https://www.npmjs.com/) ou [Yarn](https://yarnpkg.com/)

## 🔧 Instalação

1. Clone o repositório:
```bash
git clone https://github.com/RondneyLoiola/interface-expensetracker.git
```

2. Entre na pasta do projeto:
```bash
cd interface-expensetracker
```

3. Instale as dependências:
```bash
# Usando npm
npm install

# Ou usando Yarn
yarn install
```

4. Configure as variáveis de ambiente:
```bash
# Copie o arquivo de exemplo
cp .env.example .env

# Edite o arquivo .env com suas configurações
```

## 🎮 Como Usar

### Modo de Desenvolvimento

Para iniciar o servidor de desenvolvimento:

```bash
# Usando npm
npm run dev

# Ou usando Yarn
yarn dev
```

O aplicativo estará disponível em `http://localhost:5173`

### Build para Produção

Para criar uma versão otimizada para produção:

```bash
# Usando npm
npm run build

# Ou usando Yarn
yarn build
```

### Preview da Build

Para visualizar a build de produção localmente:

```bash
# Usando npm
npm run preview

# Ou usando Yarn
yarn preview
```
